User question:

Ecris moi le script d' un code Java qui créer un GUi avec un bouton.

AI answer:

Voici un exemple de code Java qui crée une interface graphique avec un bouton :

java
import javax.swing.*;
import java.awt.event.*;

public class MyGUI implements ActionListener {
     private JButton button;
     
     public MyGUI() {
          JFrame frame = new JFrame("My GUI");
          button = new JButton("Click me!");
          button.addActionListener(this);
          frame.add(button);
          frame.pack();
          frame.setVisible(true);
     }
     
     public void actionPerformed(ActionEvent e) {
          if (e.getSource() == button) {
                System.out.println("Button clicked!");
          }
     }
     
     public static void main(String[] args) {
          new MyGUI();
     }
}


Ce code crée une fenêtre avec un bouton "Click me!". Lorsque le bouton est cliqué, le texte "Button clicked!" est affiché dans la console. .
