User question:

Quelle est la différence entre le droit français et le droit anglais ?

AI answer:

Le droit français et le droit anglais ont des différences significatives en termes de sources du droit, de système juridique, de procédure judiciaire et de terminologie juridique. Par exemple, le droit français est basé sur le Code civil, tandis que le droit anglais est basé sur la common law. De plus, le système juridique français est inquisitorial, tandis que le système juridique anglais est accusatoire..
