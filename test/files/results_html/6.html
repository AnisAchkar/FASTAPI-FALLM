User question:

Tu peux me montrer du c++ concurent avec une éxecution parallèle ?

AI answer:

Voici un exemple de code C++ utilisant l'exécution parallèle pour calculer la somme des éléments d'un tableau :

c++
#include <iostream>
#include <vector>
#include <numeric>
#include <execution>

int main() {
     std::vector<int> v(1000000, 1); // tableau de 1 million d'éléments
     int sum = std::reduce(std::execution::par, v.begin(), v.end()); // calcul de la somme en parallèle
     std::cout << "La somme est : " << sum << std::endl;
     return 0;
}


Dans cet exemple, la fonction std::reduce est utilisée pour calculer la somme des éléments du tableau. L'argument std::execution::par indique que le calcul doit être effectué en parallèle. .
