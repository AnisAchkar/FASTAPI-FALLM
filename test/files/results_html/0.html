Python Code to Display Folder Contents on Tkinter GUI

Here's a Python code that opens a folder and displays its contents on a Tkinter GUI:

python
import os
import tkinter as tk
from tkinter import filedialog

root = tk.Tk()
root.withdraw()

folder_path = filedialog.askdirectory()

files = os.listdir(folder_path)

listbox = tk.Listbox(root)
listbox.pack()

for file in files:
     listbox.insert(tk.END, file)

root.mainloop()


This code first imports the necessary modules, including os for working with files and directories, and tkinter for creating the GUI. It then creates a Tkinter window and hides it using the withdraw() method.

Next, it uses the askdirectory() method from the filedialog module to prompt the user to select a folder. The selected folder path is stored in the folder_path variable.

The code then uses the listdir() method from the os module to get a list of all the files in the selected folder. It creates a Tkinter Listbox widget and adds each file name to the listbox using a for loop.

Finally, it starts the Tkinter event loop using the mainloop() method, which displays the GUI and allows the user to interact with it.
